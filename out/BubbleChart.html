Class: BubbleChart

BubbleChart(data, element, colour)

Enables the creation of word frequency bubble chart instances which support hoverable elements, different colour ranges, and changing text sizes.

Sample Data used was taken from word frequency analyses of Trump's Tweets and the Harry Potter books. Source code was inspired by the work of Alok Shukla and Elliot Bentley

Constructor

new BubbleChart(data, element, colour)

Parameters:
Name Type Description
data object Data taken is in a similar format to that of a JSON file: {"children": [{"Name": "Word", Count: 999}, {"Name": "Another Word", Count: 100}...]}.
element object The HTML element that the bubble chart will be appended to. Most common to insert into a 'div' element
colour string Choose from a selection of 'Teal', 'Orange' or 'Grey'. Other colours can be added by adapting the method setColour

addText()

Takes formatted data and creates two text elements for each word: one for count and another for the word itself. Uses the coordinates created by D3's pack method to decide where each bubble's text should go. Depending on the size of the bubble, some characters may be ommitted from the text element to stop spillage outside the bubble.
